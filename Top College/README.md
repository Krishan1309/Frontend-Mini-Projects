# University Search by Country

A simple web application that allows users to search for universities worldwide by entering a country name.  
The project fetches real-time data from a public API and displays universities in a clean and readable format.

---

## Features

- Search universities by country name  
- Real-time data from a public universities API  
- Accurate country-based filtering  
- Clean and minimal user interface  
- Responsive layout  
- Fast and lightweight (Vanilla JavaScript)

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Universities List API (Hipolabs)


---

## How It Works

1. The user enters a country name in the input field.  
2. The application sends a request to the public API.  
3. The response data is filtered and displayed dynamically on the page.  
4. If no universities are found, an appropriate message is shown.
