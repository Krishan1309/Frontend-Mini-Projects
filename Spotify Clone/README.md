# Spotify Web Player Clone (Frontend)

This project is a frontend clone of the Spotify Web Player interface built using **HTML and CSS**.  
It focuses on recreating the visual layout, structure, and styling of the Spotify desktop web player.

This is a **UI-based project** created for learning and practicing frontend development concepts.
